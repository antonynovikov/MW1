Навигация по репозиторию:

data/ - папка с датасетом

docs/ - папка с аналитической запиской

notebooks/ - основная папка с вычислительными файлами формата .ipynb

results/ - папка с визуализацией результатов
  results/figures/ - графики
  results/tables/ - таблицы

task/ - папка с требованиями к выполнению

requirements.txt - файл с зависимостями (библиотеками)
